LTH.OS(Oup.1)
LTH.OS(Oup.1) is a futuristic desktop experience built with Electron, designed to turn a standard Windows environment into a visually rich, self-contained operating-style workspace. It combines a custom desktop shell, boot flow, built-in apps, file management, browser tools, media features, LTH.editor, and a development environment called LTH PROG into a single interactive system.
This project is more than a standalone app: it is a modular desktop ecosystem with its own visual identity, window management, integrated tooling, and local-first workflow. The current 2.4.4.1 Beta focuses on security hardening, modular update foundations, refreshed M2.4 branding, and a guided full-screen "What's New in 2.4.5" experience before the user enters the desktop.
Features
Custom desktop-style interface with boot screen, taskbar, dock, and immersive visual design
Built-in window manager and modular app system
Full-screen "What's New in 2.4.5" gate before desktop entry
Integrated file explorer with configurable root paths, removable drive support, encrypted vault folders, and desktop shortcut flows
Built-in browser experience with isolated session handling
LTH PROG for project editing, preview, and development workflows
LTH.editor for local video editing and media-based workflows
LTH IA foundations for routed model workflows, live web-aware queries, and token-based usage control
Local-first architecture focused on user control, isolated storage, and portable-friendly paths
GitHub-based portable update detection and modular update groundwork
Windows-ready portable distribution support
Build flow currently focused on Windows x64
Included Core Apps
LTH Files: file explorer, navigation system, vault folders, and desktop shortcut actions
LTH Browser: integrated browsing experience
LTH PROG: development and project workspace
LTH.editor: integrated video editor for timelines, overlays, formats, and export workflows
LTH IA: assistant workspace and model-routing surface
LTH Remote: beta-locked remote surface reserved for the upcoming phone-to-PC experience
Music: media playback inside the environment
Settings: system configuration, release notes, wallpaper, security inspection, and system information
